def map_create.create_map (   url)

The main function of map create.

Calls the map scrape function and obtains the SVG from it. It then parses the SVG and converts it into a numpy array, which is then used by the path search part of the map program.

Parameters
urlthe url of the webpage (which is provided to scrape_map when it is called)
ReturnsA tuple containing two parts. The first being "label_locs", a dictionary containing the enterances and their coordinates, along with the aisles/checkouts and their coordinates. The second being "map_data", the store map as a numpy array.

Removes "blobs" from SVG to prevent overlapping shapes in the parser.

This stems from the SVG bounding boxes being used to approximate aisles, which can often overlap.

Parameters
map_datathe map, in a numpy array format
aisle_locsa dictionary of aisles labels and their respective coordinates
ReturnsA modified map_data, with the blob issue fixed.
