The heuristic function for the first A*.

This function calculates the manhattan distance between an arbitrary coordinate and the nearest aisle corrdinate. This function then checks for the number of remaining aisles, and adds that value on top of the manhattan distance. Each of these equations are calculated and weighted by importance in order to optimize the heuristic.

Parameters
xthe current x value in the current coordinate
ythe current y value in the current coordinate
aislesthe list of remaining aisles
shapethe shape of the map (used for weighting the heuristic)
checkoutsthe checkout coordinates [DEPRECIATED: NO LONGER IMPACTS THE HEURISTIC]
dxthe old x coordinate
  • [DEPRECIATED: NO LONGER IMPACTS THE HEURISTIC]
dythe old y coordinate
  • [DEPRECIATED: NO LONGER IMPACTS THE HEURISTIC]
Returnsa numerical value ranging from [0,inf). Slowly approaches 0 as the A* progresses.
