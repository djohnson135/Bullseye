App()

This react element provides the user the ability to see that they are logged in, view our logo through an image, click the Generate Path button to generate their shopping path once on the Target page, and link them to the map once it is generated. To get the user information, the handleGetProfile function is utilized from background.js. Once the Generate Path button is called, the function message is called, which utilizes the chrome.extension api to set up a message/listener with background.js where the logic to get the necessary information is written. Various react elements, html, and css styling are utlized in the design of the popup.

(async) getProductData(itemIDs, store_name) → {Set|Array|String}

For runtime listener prior to this function: Listener function utilizing chrome.extension api in order to listen in on when the user clicks the generate path button. First gets the user's profile if needed Then checks if it is a message from the React Popup. If so, runs the contenScript on the current tab Then waits until the contentScript sends data back. Calls getProductData to use RedCircle to obtain more information from the web scraped data. Finally sends data to database and to lambda to start path generation.

This function utilizes the RedCircle API with our web-scraped TCIN and store data to obtain the product information for that user. The store informaiton is pulled from our database that contains all Texas stores while everything is pulled from RedCircle.
